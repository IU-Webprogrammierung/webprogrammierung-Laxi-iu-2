
# Reisen & Fotografie – Portfolio

## Projektbeschreibung
Diese Website ist ein persönliches Portfolio mit dem Fokus auf Reisen und Fotografie. 
Ziel ist es, Reisen visuell darzustellen und übersichtlich aufzubereiten, ohne einen klassischen Blog zu verwenden.

Die Inhalte werden in einer verständlichen Struktur präsentiert, sodass Nutzer schnell Informationen erfassen können.



## Seitenstruktur

- index.html (Startseite / Onepager)
- reise1.html (Detailseite)
- reise2.html (Detailseite)
- kontakt.html (Kontaktseite)

Der Aufbau folgt einem Hybrid-Ansatz:
Onepager + separate Detailseiten.



## Technologien

- HTML5 (semantische Struktur)
- CSS3
- Flexbox (Navigation, About Section)
- CSS Grid (Reiseübersicht, Highlights)
- Media Queries (Responsive Design)
- JavaScript (Hero Counter)



## Responsive Design

Die Website ist für verschiedene Bildschirmgrößen optimiert:

- Mobile: bis 767px
- Tablet: 768px – 1279px
- Desktop: ab 1280px

Layouts passen sich entsprechend an (z.B. Grid → 1 Spalte auf Mobile).



## Barrierefreiheit

Folgende Maßnahmen wurden umgesetzt:

- Verwendung semantischer HTML-Elemente (header, main, section, footer)
- Alt-Texte für Bilder
- Labels für Formularfelder
- ausreichende Farbkontraste
- verständliche Linktexte
- einfache Navigation



## Design & Konzept

- Fokus auf Bilder (Fotografie)
- ruhiges, klares Layout
- Card-basierte Struktur für bessere Lesbarkeit
- Glassmorphism als visuelles Element
- konsistentes Spacing-System (96px / 32px)



## Besondere Features

- animierter Counter im Hero-Bereich
- horizontale Bildergalerie
- Timeline auf Detailseiten
- responsive Navigation
